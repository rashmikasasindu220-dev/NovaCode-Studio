NOVA CODE STUDIO WEBSITE - PUBLISHING GUIDE
==========================================

Files included:
- index.html
- styles.css
- script.js
- favicon.svg
- assets/nova-code-logo.svg
- assets/nova-code-icon.svg

How to open:
1. Extract the ZIP file.
2. Open index.html in Chrome, Edge, or any browser.

Before publishing, change these placeholders:
1. Email: hello@novacodestudio.com
2. Phone: +94 71 234 5678
3. Location: Colombo, Sri Lanka
4. Pricing values if needed
5. Portfolio projects when you have real client work

How to publish free on GitHub Pages:
1. Create a GitHub account.
2. Create a new repository called nova-code-studio.
3. Upload index.html, styles.css, script.js, favicon.svg, and the assets folder.
4. Go to Settings > Pages.
5. Select Deploy from branch > main > root.
6. Your website link will be created by GitHub Pages.

How to publish with a domain:
1. Buy a domain such as novacodestudio.lk or novacodestudio.com.
2. Use hosting such as GitHub Pages, Netlify, Hostinger, or a local Sri Lankan hosting provider.
3. Connect the domain DNS records to the hosting provider.

Important:
- The contact form is a demo mailto form. It opens the visitor's email app.
- For a real contact form that stores messages, you need backend hosting or a form service.
